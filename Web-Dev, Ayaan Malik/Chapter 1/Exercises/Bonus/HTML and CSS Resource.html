HTML Elements

1. Headings

Use h1 to h6 for titles

<h1>Main Title</h1>

2. Paragraph

For regular text

<p>Some text.</p>

3. Links

Connect pages

<a href="page.html">Click</a>

4. Images

Show pictures

<img src="pic.jpg" alt="Photo">

5. Lists

Bullet points

<ul>
  <li>Item 1</li>
</ul>
